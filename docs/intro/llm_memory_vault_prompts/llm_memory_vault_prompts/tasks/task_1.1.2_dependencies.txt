### 2. Install Dependencies

After creating the file, run:
```bash
poetry install
```

This should:
- Create a virtual environment
- Install all dependencies
- Make the `vault` command available

### 3. Verify Installation

Test that everything works:
```bash
# Activate virtual environment
poetry shell

# Check CLI entry point exists
which vault
vault --version  # Should work even if command not implemented yet

# Test imports
python -c "import vault; print(vault.__version__)"
python -c "import typer; import sqlalchemy; import cryptography; print('✓ Core deps OK')"
```

## Key Dependencies Explained

### Core Framework
- **fastapi**: REST API (Phase 5)
- **pydantic**: Data validation and settings
- **typer**: CLI framework with rich formatting

### Security
- **cryptography**: Fernet encryption for blobs
- **argon2-cffi**: Password hashing for key derivation

### Storage
- **sqlalchemy**: ORM for SQLite
- **qdrant-client**: Vector database (Phase 4)

### Processing
- **sentence-transformers**: Local embeddings
- **openai**: API client (compatible with Ollama)

### Dev Tools
- **black**: Code formatting (opinionated)
- **ruff**: Fast linting (replaces flake8, isort)
- **mypy**: Type checking
- **pytest**: Testing framework

## Safety Constraints

## Common Issues & Solutions

**Issue**: Poetry install fails with resolver error
**Solution**: Clear cache `poetry cache clear pypi --all` and retry

**Issue**: Python version not found
**Solution**: Use pyenv to install Python 3.11+

**Issue**: M1/M2 Mac compilation errors
**Solution**: Set `export ARCHFLAGS="-arch arm64"` before installing
